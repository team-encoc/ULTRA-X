## 6. 탈퇴 시 주의사항

### 6.1 Fish Finder 기기 연결 해제

**참고**: Fish finder는 WiFi/Bluetooth로 앱과 직접 연결되며, 서버에 연결 정보가 저장되지 않습니다. 앱 삭제 또는 계정 탈퇴 시 자동으로 연결이 해제되며, 다른 기기에서 재연결하여 사용할 수 있습니다.

### 6.2 어탐 데이터 백업

**중요**: 탈퇴 전 어탐 CSV 데이터를 직접 다운로드 받으시기 바랍니다.

**데이터 다운로드 방법:**

1. ULTRAX 앱 실행
2. **설정** > **낚시 기록** > **데이터 내보내기**
3. 내보낼 데이터 선택 (어탐 CSV 파일, 낚시 스폿 위치 정보)
4. 다운로드 위치 선택 (기기 저장소)

**참고**:

- 본 서비스는 사진/비디오 업로드 기능을 제공하지 않습니다.
- 어탐 데이터는 CSV 파일 형식으로만 저장됩니다.

**주의**: 탈퇴 후에는 백업 데이터에 접근할 수 없으며, 모든 데이터가 영구 삭제됩니다.

### 6.3 소셜 로그인 연동 해제

**소셜 로그인으로 가입한 경우:**

- **네이버 로그인**: 네이버 계정 > 보안 설정 > 외부 사이트 로그인 관리 > ULTRAX 연동 해제
- **카카오 로그인**: 카카오톡 > 설정 > 카카오계정 > 연결된 서비스 관리 > ULTRAX 연결 끊기
- **구글 로그인**: Google 계정 > 보안 > 타사 액세스 권한이 있는 앱 > ULTRAX 액세스 권한 삭제

**중요**: 소셜 로그인 연동 해제는 자동으로 처리되지 않으므로, 탈퇴 후 수동으로 해제하는 것을 권장합니다.

### 6.4 앱 삭제와 탈퇴의 차이

| 구분             | 앱 삭제        | 계정 탈퇴      |
| ---------------- | -------------- | -------------- |
| 계정 유지        | ✅ 유지        | ❌ 삭제        |
| 데이터 보관      | ✅ 서버에 보관 | ❌ 영구 삭제   |
| 재설치 시        | ✅ 로그인 가능 | ❌ 로그인 불가 |
| Fish finder 연결 | ✅ 유지        | ❌ 해제        |

**중요**: 앱을 삭제해도 계정은 유지되며, 재설치 후 로그인하면 기존 데이터를 사용할 수 있습니다. 계정을 완전히 삭제하려면 반드시 **계정 탈퇴** 절차를 진행해야 합니다.

## 8. 탈퇴 관련 자주 묻는 질문 (FAQ)

### Q1. 탈퇴하면 바로 계정이 삭제되나요?

**A1.** 아니요. 탈퇴 신청 후 **7일간의 유예기간**이 제공됩니다. 이 기간 동안 로그인하여 탈퇴를 취소할 수 있습니다. 유예기간 경과 후 모든 데이터가 영구 삭제됩니다.

### Q2. 유예기간 중 서비스를 이용할 수 있나요?

**A2.** 유예기간 중에는 로그인은 가능하지만 서비스 이용은 제한됩니다 (읽기 전용 모드). Fish finder 연결, 낚시 스폿 저장 등의 기능은 사용할 수 없습니다.

### Q3. 앱을 삭제하면 계정도 삭제되나요?

**A3.** 아니요. 앱 삭제는 기기에서 앱만 제거하는 것이며, 서버에 있는 계정과 데이터는 유지됩니다. 계정을 완전히 삭제하려면 반드시 **계정 탈퇴** 절차를 진행해야 합니다.

### Q4. 탈퇴 후 재가입할 수 있나요?

**A4.** 네. 다만 동일 이메일로는 탈퇴 후 **15일간 재가입이 제한**됩니다. 다른 이메일 주소를 사용하면 즉시 재가입 가능합니다.

### Q5. 재가입하면 탈퇴 전 데이터가 복구되나요?

**A5.** 아니요. 탈퇴 후 삭제된 데이터는 복구되지 않습니다. 재가입 시 새로운 계정으로 시작해야 합니다.

### Q6. Fish finder 기기는 어떻게 되나요?

**A6.** Fish finder는 WiFi/Bluetooth로 앱과 직접 연결되며, 서버에 연결 정보가 저장되지 않습니다. 계정 탈퇴 또는 앱 삭제 시 자동으로 연결이 해제되며, 언제든지 다른 기기에서 재연결하여 사용할 수 있습니다.

### Q7. 법령에 따라 보관되는 데이터는 무엇인가요?

**A7.** 계약/청약철회 기록(5년), 소비자 불만/분쟁처리 기록(3년), 부정 이용 기록(1년), 접속 로그(6개월) 등이 법령에 따라 일정 기간 보관 후 삭제됩니다.

### Q8. 소셜 로그인 연동은 자동으로 해제되나요?

**A8.** 아니요. 네이버, 카카오, 구글 로그인 연동은 각 소셜 플랫폼에서 수동으로 해제해야 합니다.

### Q9. 탈퇴 취소는 어떻게 하나요?

**A9.** 유예기간 중 ULTRAX 앱에 로그인한 후 **"탈퇴 취소"** 버튼을 클릭하면 즉시 계정이 복구됩니다.

### Q10. 탈퇴 후 데이터 삭제 확인을 받을 수 있나요?
