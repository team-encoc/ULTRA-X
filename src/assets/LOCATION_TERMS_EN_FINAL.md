## Article 3-2 (Details of Background Location Services)

### 1. Purpose of Background Location Collection

The Company collects location information in the background for the following purposes:

**Primary Purpose:**

- **Maintain Continuous Connection with Fish Finder Device**

  - Maintain Bluetooth/WiFi connection with fish finder even when app is switched to background during fishing
  - Provide uninterrupted real-time sonar data
  - Ensure inter-device communication stability

- **Automatic Fishing Route and Location Recording**

  - Automatically track and save movement routes during fishing
  - Map location-specific sonar data (record where fish were detected)
  - Improve accuracy of fishing records

- **Location-based Safety Alerts**
  - Real-time warnings when approaching hazardous areas (deep water zones, rocky areas)
  - Alerts when entering restricted zones (fishing prohibited areas, protected areas)

### 2. User Value

Background location services provide the following value:

- **Uninterrupted Fishing Experience**: Maintain connection with fish finder even when switching app to background or using other apps
- **Automatic Recording**: Automatically record location and sonar data during fishing, eliminating need for manual saving
- **Enhanced Safety**: Real-time alerts when approaching hazardous areas improve fishing safety
- **Battery Optimization**: Efficient location collection algorithms minimize battery consumption

### 3. Location Collection Frequency and Battery Optimization

The Company optimizes location collection to minimize user battery consumption as follows:

- **Collection Frequency**: Background location collection only when connected to fish finder (average 30-second intervals)
- **Adaptive Collection**: Automatically adjust collection frequency based on movement speed
  - When stationary: Reduced collection frequency (1-minute intervals)
  - When moving: Increased collection frequency (10-30 second intervals)
- **Stop on Disconnection**: Immediately stop background location collection when fish finder connection is terminated
- **Battery Saver Mode**: Automatically reduce location collection frequency when battery is below 20%

### 4. How to Disable Background Location

Users can disable background location collection at any time:

#### Android

1. Device Settings > Apps > ULTRAX
2. Permissions > Location
3. Select **"Allow only while using the app"**

#### iOS

1. Device Settings > ULTRAX
2. Location
3. Select **"While Using the App"**

**Important**: Disabling background location will:

- May disconnect fish finder connection when app is in background
- Limit automatic fishing route recording
- Prevent real-time safety alerts

### 5. Background Location Data Retention and Deletion

- **Retention Period**: Until fishing spot is saved (immediately deleted if user deletes)
- **Automatic Deletion**: Route data not saved by user is automatically deleted after session ends
- **Manual Deletion**: App Settings > Fishing Records > Delete Route Data
